# Тестовое задания для компании "Idea Platform"

Данные берутся с .json файла, интересно было оптимизировать под IE 11, до этого подобным не занимался.

Стэк:
- React + Redux TLK;
- Typescript;
- Scss;
- Vite;
- Axios;

## Запуск проекта:
Если не установлен yarn:
```
npm i -g yarn
```
Далее:
```
yarn
```
```
yarn dev
```
